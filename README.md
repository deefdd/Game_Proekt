# Game_Proekt
Цей проект стоворений на базі Python, який буде допомагати краще та спокійніше грати ігри на слабих та середніх компютерах
#Функціонал 
 - Оптимізує компютер під певну гру та під ваші комплектуючі
 - стабіліцує FPS та плавність картинки
 - оптимізує та розганяє компоненти при потребі
#Використані техноголії
 - Python 3
 - також такі бібліотеки як: psutil, subprocess, ctypes
